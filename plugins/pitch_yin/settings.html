YIN Pitch Detector

Real-time pitch detection via a C++ WebAssembly YIN algorithm. Select your microphone, hit Test mic to see it live, then use M in the player to toggle detection during play-along.

Live preview

Lower = more detections (including uncertain ones). Higher = only very clean pitches get through.

Passes the raw mic signal through to your speakers so you can hear yourself play. Starts at 0 to avoid feedback — raise it only when using headphones or a direct monitor. Also adjustable in the player's Mixer popover while playing.

⚠ Raise above 0 only when using headphones — speakers will cause feedback.
